== Description ==

Client Handover Pro turns "the website is done" into a repeatable, professional handover process.

**The workflow**

1. Build the website as usual (Elementor, Bricks, WooCommerce, or plain WordPress).
2. Install Client Handover Pro.
3. Run the Launch Checklist — it automatically inspects SSL, forms, SEO, performance basics, and security.
4. Fix anything flagged, using the built-in links to the relevant settings screen.
5. Generate the Client Dashboard — a simplified admin experience for the client's role.
6. Build a short Tutorial Center so the client can make their own edits with confidence.
7. Export a PDF Handover Document and get client sign-off.

== Installation ==

1. Upload the `client-handover-pro` folder to `/wp-content/plugins/`.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Open "Handover Pro" in the admin menu and click **Run Scan**.

== Frequently Asked Questions ==

= Does this replace my SEO or caching plugin? =

No. The Launch Checklist detects whether a caching, SEO, SMTP, or security plugin is already active and reports on it — it doesn't replace those plugins.

= Is the PDF export a real PDF file? =

The Handover Document opens as a print-ready page; use your browser's Print → Save as PDF to export it. No server-side PDF library is required.

= Where are credentials stored? =

The Credentials Vault (Pro) encrypts values with AES-256-CBC using a key derived from your site's WordPress salts, and the page is only ever rendered for administrators.
